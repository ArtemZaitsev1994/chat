# Логика работы 

Соц. сеть с регистрацией пользователей, которые создают группы (компании) для общения с друзьями. 

## На данном этапе подключен:
* redis для связи между микросервисами
* golang для чата и сокетов
* python для отдачи статики и основная логика работы. 
* nginx как прокси
* база данных - MongoDB
